# Expense Tracker - FastAPI Prototype

A simple **Expense Tracker** prototype built with **FastAPI**.  
This project allows users to manage and track expenses through a RESTful API. It is designed as a starting point for a full-featured expense tracking application.

---

## Features

- Add, view, update, and delete expenses
- Categorize expenses (e.g., Food, Transport, Entertainment)
- RESTful API endpoints using FastAPI
- Lightweight and fast prototype suitable for further expansion

---

## Tech Stack

- **Backend:** FastAPI
- **Database:** SQLite / PostgreSQL (can be configured)
- **Python Version:** 3.10+
- **Dependencies:** See `requirements.txt`
